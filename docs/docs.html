Guía de Inicio

Documentación de Iridium

Bienvenido a la documentación oficial de Iridium Language. Este lenguaje ha sido diseñado desde cero para ser la columna vertebral de aplicaciones backend que requieren alta resiliencia y procesamiento de inteligencia artificial nativa.

¿Por dónde empezar?

Si eres nuevo, te recomendamos seguir la ruta del Nivel 1 (El Despegue) para entender cómo instalar el motor e interpretar tus primeros archivos .iri.

Filosofía del Lenguaje

Híbrido por diseño

Flexibilidad de Python con la seguridad de tipos necesaria para entornos de datos masivos.

Resiliencia Nativa

El backend no se detiene. Los fallos de red se gestionan con elegancia usando el operador Sponge.
