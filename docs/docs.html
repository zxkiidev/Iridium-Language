Documentación

Aprende a dominar Iridium, el lenguaje diseñado para administrar backends de nueva generación con inteligencia artificial integrada.

Instalación

Para ejecutar archivos .iri, necesitas el motor de ejecución basado en Python 3.10+.

// Clona el repositorio oficial
git clone https://github.com/zxkiidev/Iridium-Language.git

// Ejecuta el motor
python src/main.py examples/test.iri

Variables y Tipado

Iridium permite un tipado híbrido. Puedes ser estricto para asegurar la integridad de la IA o flexible para prototipado rápido.

var usuario: string = "Zadquiel"
var precision: float = 0.98
const ID_NUCLEO = 77

Seguridad (Secure)

Los bloques secure: cifran visualmente cualquier salida de datos sensibles, ideal para proteger claves de API o contraseñas en el backend.

secure:
    var password = "clave_secreta_77"
    out "Clave: {password}" // Resultado: ********
end

Operadores de Resiliencia

El operador Sponge ~> permite que el código no se detenga si una conexión externa falla, asignando un valor de respaldo.

// Si la API falla, conexion_ia será 0.0
var conexion_ia = network.get("api.ia.com") ~> 0.0
out "Estado: {conexion_ia}"
