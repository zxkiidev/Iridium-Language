Documentación de Iridium

Bienvenido a la documentación oficial de Iridium Language. Este lenguaje ha sido diseñado desde cero para ser la columna vertebral de aplicaciones que requieren alta resiliencia y procesamiento de IA nativa.

Hoja de Ruta Académica

Si eres nuevo, te recomendamos seguir la ruta del Nivel 1 (El Despegue) para entender la filosofía del lenguaje antes de saltar a la lógica de Tensores para IA.

Filosofía del Lenguaje

Híbrido por diseño

Flexibilidad de Python con la seguridad de tipos necesaria para entornos de datos masivos.

Resiliencia Nativa

El backend no se detiene. Los fallos se gestionan con elegancia usando el operador Sponge (~>).
