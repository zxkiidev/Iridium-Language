Tu primer script: Hola Mundo

Todo gran viaje en la programación comienza con un "Hola Mundo". En Iridium, la simplicidad es nuestra prioridad para que puedas centrarte en la lógica de datos.

1. Crear el archivo

Abre tu editor de texto favorito (recomendamos VS Code con nuestra extensión oficial) y crea un nuevo archivo llamado main.iri.

2. Escribir el código

Escribe la siguiente línea de código en tu archivo:

out("Hola Mundo desde Iridium!")

3. Ejecución desde la terminal

Para ejecutar tu script, abre la terminal en la carpeta donde guardaste el archivo y usa el comando del motor:

> iridium main.iri
Hola Mundo desde Iridium!

4. ¿Qué está pasando?

Cuando ejecutas este comando, ocurren tres cosas en milisegundos:

  • Lectura: El motor busca el archivo main.iri.
  • Análisis (Parsing): Identifica la función out() como una instrucción de salida estándar.
  • Ejecución: Envía la cadena de texto a la consola del sistema.

Tip Pro

En Iridium, no necesitas puntos y comas (;) al final de las líneas. Queremos que tu código sea tan limpio como tu pensamiento.
