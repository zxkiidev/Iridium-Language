Tensores y Computación Numérica

Iridium incorpora el tipo de dato tensor en su núcleo, permitiendo realizar operaciones matemáticas de alta velocidad sobre grandes volúmenes de datos sin necesidad de importar librerías externas como NumPy.

1. Declaración de Tensores

Un tensor puede ser una matriz simple o una estructura multidimensional. Se declaran usando la palabra clave tensor.

// Creación de una matriz 2x2
var pesos: tensor = {{0.1, 0.9}, {0.4, 0.2}}

2. Operaciones Vectorizadas

A diferencia de las listas normales, los tensores en Iridium soportan aritmética directa. Esto es fundamental para los algoritmos de redes neuronales.

var a = tensor({1, 2})
var b = tensor({3, 4})
var c = a * b // Resultado: {3, 8}

3. Integración con el Motor de IA

Los tensores son la moneda de cambio entre el programador y el motor de IA de Iridium. Cualquier dato procesado puede ser inyectado directamente en un modelo pre-entrenado.

var input_data = myDB.fetchTensor("SELECT pixels FROM images")
var prediccion = ia.run(model, input_data)

Paralelismo

En la arquitectura de Iridium, las operaciones con el tipo tensor se delegan automáticamente a hilos de ejecución paralelos, aprovechando todos los núcleos de tu CPU de manera transparente.
