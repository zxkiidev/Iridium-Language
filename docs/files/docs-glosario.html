Glosario Técnico

Referencia rápida de todos los términos, palabras clave y operadores del lenguaje Iridium. Ordenados alfabéticamente para consulta ágil.

await

Captura de entrada del usuario.

Pausa la ejecución hasta que el usuario introduce un valor. Compatible con tipado: var edad: int = await "Edad: "

avrg()

Función de media aritmética.

Calcula el promedio de una colección o dataset. Disponible globalmente sin imports.

check_password()

Verificación de contraseña hasheada.

Compara una contraseña en texto plano contra su hash almacenado. Retorna true o false.

class

Definición de clase y objeto.

Permite crear tipos personalizados con campos (var) y métodos (function). Se cierra con end.

const

Variable inmutable.

Una vez asignada, no puede modificarse. Reasignarla provoca Ir-Error: Inmutable violation y detiene la ejecución.

cronos

Tipo de dato temporal.

Variable especial para datos que se actualizan periódicamente (sensores, streams). Planificado para implementación completa en V2.

dataset

Estructura de datos para IA.

Colección inmutable y comprimida en memoria, optimizada para operaciones matemáticas vectorizadas. Compatible con avrg().

db

Objeto global de base de datos.

Métodos: db.config(), db.sqlite(), db.query(), db.query_one(), db.exec(), db.create_table(), db.init().

end

Cierre de bloque explícito.

Cierra los bloques: while, for, switch, secure, class, function, route.

env

Cargador de variables de entorno.

env of ".env": carga el archivo al inicio. Las variables se mapean a constantes con FROM env.file.

function

Definición de método en clase.

Sintaxis: function nombre(param1, param2):. Solo válido dentro de un bloque class. Se cierra con end.

hash_password()

Hashing seguro de contraseñas.

Genera un hash seguro (Werkzeug/PBKDF2) de un texto. Úsalo antes de guardar contraseñas en la base de datos.

html()

Respuesta HTML desde una ruta.

Retorna una respuesta HTTP con Content-Type: text/html. Acepta código de estado opcional: html(contenido, 404).

ia

Objeto global de inferencia.

Expone ia.predict(tensor, model_id) para ejecutar predicciones. El motor gestiona el threading de inferencia internamente.

jsonify()

Respuesta JSON desde una ruta.

Serializa un objeto a JSON con Content-Type: application/json. Acepta estado HTTP: jsonify({error: "msg"}, 401).

math

Objeto global de matemáticas.

Proporciona: math.sqrt(), math.pow(), math.floor(), math.ceil(), math.abs(), math.pi.

out

Instrucción de salida estándar.

Imprime en consola con interpolación de variables mediante {nombre}. Dentro de secure:, los valores se enmascaran como ********.

route

Declaración de endpoint HTTP.

Sintaxis: route "/ruta" => "MÉTODO":. Inyecta automáticamente body, params y path_params. Se cierra con end.

secure

Bloque de seguridad.

Todo lo ejecutado dentro activa encriptación en tránsito y ofusca las salidas out. Las variables se limpian de RAM al salir del bloque.

serve()

Inicia el servidor HTTP.

Lanza el servidor en el puerto indicado: serve(8080). Bloquea la ejecución hasta Ctrl+C. Gestiona multi-threading automáticamente.

tensor

Tipo de dato matricial.

Matrices multidimensionales con aritmética directa (a * b opera elemento a elemento). Las operaciones se paralelizan automáticamente en CPU.

var

Declaración de variable.

Modo dinámico por defecto. Puede tipearse: var x: int = 5. Reasignable en cualquier momento.

~>

Operador Sponge (valor de respaldo).

Si la expresión de la izquierda falla o lanza error, asigna el valor de la derecha. Evita excepciones en operaciones externas.

>>

Operador Pipe (redirección).

Redirige la salida de una expresión. Uso actual: variable >> out para imprimir el valor directamente al log de depuración.
