Entrada y Salida (I/O)

La comunicación entre el programa y el usuario es vital. Iridium simplifica esto mediante una sintaxis limpia para imprimir datos y una instrucción asíncrona para recibirlos.

1. Salida de Datos: `out`

La instrucción out envía información a la consola. Soporta interpolación nativa de variables usando llaves {}.

var usuario = "zxkiidev"
out "Bienvenido al sistema, {usuario}"

Puedes incluir expresiones matemáticas o llamadas a funciones directamente dentro de las llaves:

out "El resultado es: {math.sqrt(64)}"

2. Entrada de Datos: `await`

Para capturar información del usuario, utilizamos la palabra clave await. Esta pausa la ejecución del motor hasta que el usuario presiona Enter.

// Captura interactiva
var edad: int = await "Ingresa tu edad: "

out "Tienes {edad} años."

3. Flujos de Datos Dinámicos

En el contexto de Ciencia de Datos, Iridium permite conectar entradas y salidas mediante el operador Pipe (>>).

// Redirección de salida a log de depuración
var accuracy = 0.95
accuracy >> out // Mirror Log activo

Seguridad I/O

Recuerda que si estás dentro de un bloque secure:, los datos impresos con out se mostrarán como asteriscos (********) a menos que especifiques lo contrario.
