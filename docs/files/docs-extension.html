Extensión Oficial para VS Code

La extensión oficial de Iridium para Visual Studio Code transforma tu editor en un entorno completo para desarrollar scripts .iri. Cero configuración, funciona desde el primer archivo que abras.

1. Instalación

Puedes instalar la extensión directamente desde el Marketplace de VS Code o desde la página oficial del proyecto:

  • Abre VS Code y ve a la pestaña Extensiones (Ctrl+Shift+X).
  • Busca "Iridium Language" en el buscador.
  • Haz clic en Instalar. No requiere reinicio.

Alternativamente, puedes instalarla desde la página de la extensión pulsando el botón "VS Code Extension" en la barra de navegación superior.

2. Resaltado de Sintaxis

La extensión reconoce automáticamente los archivos .iri y aplica un esquema de colores que distingue visualmente:

const UMBRAL: float = 0.85  // Palabras clave → azul
var nombre = "Iridium"        // Strings → naranja
var epoca: int = 100        // Tipos → verde agua
avrg(datos)                 // Funciones → amarillo

3. Autocompletado Inteligente

Al escribir en un archivo .iri, la extensión sugiere automáticamente:

  • Todas las palabras clave del lenguaje (var, const, route, secure, etc.)
  • Los métodos del objeto db: al escribir db. aparecen query, exec, query_one, etc.
  • Los métodos de math e ia de forma contextual.
  • Las variables definidas en el mismo archivo.

4. Ejecución con Atajo de Teclado

Una vez instalada la extensión y configurado el motor en el PATH, puedes ejecutar el archivo .iri activo directamente desde VS Code:

# Atajo de teclado (configurable en keybindings.json):
Ctrl + Shift + B   →  iridium main.iri
🚀 [Iridium Web] Escuchando en http://0.0.0.0:8080

5. Snippets Disponibles

La extensión incluye snippets de código para las estructuras más comunes. Escribe el prefijo y pulsa Tab:

Prefijo Genera
iroute Bloque route "/ruta" => "GET": ... end
irclass Estructura de class Nombre(): ... end
irsecure Bloque secure: ... end
irenv Bloque env of ".env": ...
irapi Plantilla completa de API con db, route y serve

Compatibilidad

La extensión es compatible con VS Code 1.80 o superior en Windows, macOS y Linux. Se actualiza automáticamente junto con el motor principal de Iridium.
