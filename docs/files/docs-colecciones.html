Colecciones y Datasets

Iridium redefine la forma de trabajar con grupos de datos. A diferencia de otros lenguajes, las colecciones en Iridium están optimizadas para el flujo de entrenamiento de modelos de IA.

1. Arreglos Estándar

Se definen utilizando llaves {}. Pueden contener cualquier tipo de dato si se usa var, o tipos específicos si se declara el tipado.

// Arreglo dinámico
var lista = {1, 2, "Iridium", true}

// Arreglo de tipo estricto
var notas: list[int] = {10, 20, 30}

2. El Tipo `dataset` (IA Ready)

El tipo dataset es una estructura inmutable por defecto y altamente comprimida en memoria, diseñada para ser procesada por el núcleo de Ciencia de Datos de Iridium.

var entrenamiento: dataset = {0.11, 0.77, 0.56, 0.89}
var media = avrg(entrenamiento)
out "Media del dataset: {media}"

3. Rangos Simplificados

Para generar secuencias rápidas (útiles en bucles de entrenamiento), Iridium utiliza la sintaxis de rango nativo [inicio..fin].

var epocas = [1..100]
// Esto genera automáticamente una colección del 1 al 100

Rendimiento en IA

A diferencia de las listas normales, los dataset se cargan directamente en los registros del motor de tensores, permitiendo operaciones matemáticas vectorizadas.
