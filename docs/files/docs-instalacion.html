Instalación Avanzada

Configurar Iridium es un proceso diseñado para ser transparente pero potente. Nuestro instalador oficial para Windows gestiona la arquitectura del sistema para que puedas programar sin preocuparte por las rutas.

1. ¿Qué hace el instalador?

Cuando ejecutas Iridium_Installer_x64.exe, el motor realiza las siguientes operaciones técnicas:

  • Directorio Raíz: Crea la carpeta C:\Program Files\Iridium donde se aloja el ejecutable núcleo.
  • Registro de PATH: Añade automáticamente la ruta al sistema mediante HKLM\SYSTEM\CurrentControlSet. Esto permite usar el comando iridium en cualquier consola.
  • File Association: Registra la extensión .iri en el registro de Windows para que el sistema reconozca tus scripts.

2. Verificación por Terminal

Para asegurar que la instalación fue exitosa, abre CMD o PowerShell y escribe:

> iridium --version
Iridium Engine v3.8 | Build: 2026.02.14

3. Solución de Problemas (Troubleshooting)

Aviso: Si después de instalar recibes el error "iridium no se reconoce como un comando", asegúrate de haber reiniciado tu terminal para que Windows cargue las nuevas variables de entorno.

4. Instalación Manual

Para usuarios avanzados o entornos portables, puedes simplemente descargar el .zip, extraerlo y añadir la ruta a tus variables de entorno manualmente desde el Panel de Control.
