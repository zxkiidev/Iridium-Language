Integración con IA Nativa

Iridium incorpora un objeto ia disponible de forma global en cualquier script .iri. No necesitas importar ninguna librería: el motor de inferencia está integrado directamente en el núcleo del lenguaje.

1. El Objeto Global `ia`

El objeto ia expone los métodos de inferencia del motor. Su función principal es ia.predict(), que toma un tensor de entrada y un identificador de modelo.

// Predicción básica con datos en memoria
var entrada: tensor = {{0.5, 0.3}, {0.8, 0.1}}
var resultado = ia.predict(entrada, 1)
out "Predicción del modelo: {resultado}"

2. Predicción desde Base de Datos

El flujo más habitual en producción consiste en obtener datos de la base de datos directamente como tensores y pasarlos al motor. El método db.fetchTensor() realiza esta conversión automáticamente.

db.config("localhost", "root", "", "mi_base")

// Obtener datos de imágenes como tensor
var input_data = myDB.fetchTensor("SELECT pixels FROM images WHERE set = 'test'")
var prediccion = ia.predict(input_data, 42)

out "Clase detectada: {prediccion}"

3. Integración con Endpoints Web

Puedes combinar el motor de IA con el sistema de rutas para construir una API de inferencia con pocas líneas de código:

route "/predecir" => "POST":
    var datos: tensor = body.datos
    var modelo_id = body.modelo_id ~> 1
    var res = ia.predict(datos, modelo_id)
    return jsonify({resultado: res, ok: true})
end

serve(8080)

4. El Objeto `math` para Preprocesado

Antes de alimentar datos a un modelo, es común normalizar o transformar valores. Iridium expone el objeto global math con las funciones más usadas en pipelines de datos:

var raiz = math.sqrt(144) // → 12.0
var potencia = math.pow(2, 10) // → 1024.0
var inferior = math.floor(3.9) // → 3
var superior = math.ceil(3.1) // → 4
var pi_val = math.pi // → 3.14159...

Roadmap: Modelos Personalizados

En la V2 de Iridium, el objeto ia permitirá cargar modelos desde archivos .irimodel propios. La V3 incluirá entrenamiento distribuido directamente desde el lenguaje, sin depender de frameworks externos.
