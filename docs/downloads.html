Hazlo realidad.

Descarga el instalador oficial de Iridium y comienza a construir el futuro del backend.

Descargar para Windows

Todo en uno

El instalador configura el motor de Python, añade Iridium al PATH y prepara las dependencias.

Requisitos

Windows 10/11, Python 3.10+ y 50MB de espacio libre en disco.

Código Limpio

Sin software adicional. Solo las herramientas de Iridium para desarrolladores.

Nota: Al ejecutar el instalador, asegúrate de marcar la casilla "Add Iridium to PATH" para poder usar el comando iridium en cualquier terminal.
